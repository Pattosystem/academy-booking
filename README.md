# VANDS & LIAS Medical Academy 予約システム

医療従事者向け学習プログラム「VANDS & LIAS Medical Academy」の予約・運営システム。

## URL 構造

| 用途 | パス | 説明 |
|---|---|---|
| 申込フォーム | `/` | 参加者がアクセスする入口。紹介者URL `?ref=xxx` 対応 |
| 予約管理ページ | `/manage.html?token=xxx` | 申込確定後、参加者本人が予約変更・キャンセルする画面 |
| 管理画面 | `/admin.html` | KOTAとVands側スタッフが運営する司令塔(本番ではAuth必須) |

## 現在のステータス

**プロトタイプ段階(モックデータ)**

このバージョンはUI動作確認用のプロトタイプです。すべての画面はモックデータで動作し、申込内容は実際には保存されません。本番運用には以下の作業が必要です:

- Supabase プロジェクトの作成と `schema.sql` の実行
- 各HTMLファイルへの Supabase クライアント接続
- 確定通知メールの自動送信(Supabase Edge Function)
- 管理画面への認証(Supabase Auth)

## 主な機能

### 申込フォーム
- 3画面構成(基本情報 → スケジュール選択 → 確認)
- 3コマ × 3教室の9セッションから残席表示付きでセルフ予約
- 紹介者URLによるトラッキング(`?ref=xxx`)
- 申込確定時に予約管理用トークンURLを発行

### 予約管理ページ(参加者向け)
- 連絡先・備考の更新
- 受講スケジュールの変更
- キャンセル(開催10日前まで)

### 管理画面(運営者向け)
- ダッシュボード(申込状況・開催/キャンセル期限カウントダウン・職種内訳・紹介者ランキング・各セッション充足率)
- 申込者一覧(検索・フィルタ・CSV出力・ステータス変更)
- コマ別ビュー(各セッションの参加者リスト)
- 紹介者URL発行・管理
- イベント情報・セッション内容の自社編集機能
